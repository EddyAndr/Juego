# Juego
Juego: Adivina el número
Instrucción: Adivina el número (entre 1 y 5) con una única pista. Puedes elegir entre tres niveles.

Informacion sobre la estructura del juego:
El juego comienza con una introducción y una instrucción.
El usuario selecciona un nivel (1, 2 o 3).
Dependiendo del nivel seleccionado, se proporciona una pista diferente.
El usuario intenta adivinar el número basado en la pista proporcionada.
Si el usuario adivina correctamente, se muestra un mensaje de éxito; de lo contrario, se le pide que intente de nuevo.
